a) VirtualBoxin asennus

Aluksi menin verkkosivulle virtualbox.org. VirtualBoxin verkkosivulta asensin VirtualBox 7.2.4 platform packages – Windows hosts -version. Asennuksen aikana tuli esiin vaihe “Missing dependencies”, jossa asennus ilmoitti Python Core Package -komponentin puuttumisesta. Tästä huolimatta VirtualBoxin asennus sujui ilman muita vaikeuksia.

Linuxin asennus

Laitteena käytin TUF Gaming FX505DT_FX505DT -tietokonetta. Käyttöjärjestelmänä oli Microsoft Windows 11 Home. VirtualBoxissa painoin Machine → New.

Tämän jälkeen hain Googlesta hakusanalla “Debian live iso” ja siirryin debian.org-verkkosivulle. Latasin sivulta Live GNOME -linkistä ISO-tiedoston.

Asetin virtuaalikoneelle Base memoryksi 4000 MB ja suorittimien määräksi 4. Virtuaaliselle kiintolevylle annoin tilaa 40 GB. Tämän jälkeen painoin Finish ja loin virtuaalikoneen.

Menin virtuaalikoneen Storage-asetuksiin ja liitin lataamani Debian ISO -tiedoston virtuaaliseksi CD-ROM-levyksi.

Käynnistys

Käynnistin virtuaalikoneen, mutta käynnistys epäonnistui. Virheilmoitus oli “Not in a hypervisor partition – AMD-V is disabled in the BIOS”. Tämän vuoksi siirryin laitteen BIOS-asetuksiin ja otin käyttöön SVM-mode-asetuksen.

Kokeilin käynnistää virtuaalikoneen uudelleen, ja se käynnistyi onnistuneesti. Valitsin käynnistysvalikosta “Live system (amd64)”.

Tämän jälkeen näyttöön ilmestyi virheilmoitus: “This kernel requires an x86-64 CPU, but only detected an i686 CPU”. Korjasin ongelman menemällä virtuaalikoneen asetuksiin ja vaihtamalla käyttöjärjestelmän version 32-bittisestä 64-bittiseksi.

Asetuksen muuttamisen jälkeen virtuaalikone käynnistyi ongelmitta, ja Linux-käyttöjärjestelmä saatiin käynnistettyä onnistuneesti.
